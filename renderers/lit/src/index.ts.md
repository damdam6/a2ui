# renderers/lit/src/index.ts

## 개요

`@a2ui/lit` 패키지의 최상위 진입점 파일이다. v0.8 모듈 전체를 `v0_8` 네임스페이스로 re-export하는 단일 역할만 담당한다. 파일 상단의 JSDoc에 `@deprecated` 주석으로 신규 코드에서는 `@a2ui/lit/v0_8` 경로를 직접 사용하도록 권장하고 있다. 하위 호환성을 위해 유지된다.

## 의존성

없음

- [`./0.8/index.js`](./0.8/index.ts.md) — v0.8 전체 모듈 (`v0_8` 네임스페이스로 re-export)

## 동작 흐름

파일 로드 시 `./0.8/index.js`의 모든 공개 심볼을 `v0_8` 이름 아래 노출한다. 소비자는 `import { v0_8 } from '@a2ui/lit'` 형태로 사용할 수 있다. 신규 코드에서는 `@a2ui/lit/v0_8`로 직접 import하는 것이 권장된다.
