# renderers/web_core/src/v0_8/styles/shared.ts

## 개요

스타일 시스템 전반에서 공유되는 기초 설계 상수를 정의하는 파일이다. 현재는 그리드 단위(grid unit) 단 하나의 상수만 내보낸다. 다른 스타일 모듈들이 일관된 간격·크기 계산의 기준으로 이 값을 참조하도록 설계되어 있다.

## 의존성

- 외부 패키지: 없음
- 저장소 내부 모듈: 없음

## Exports

| 이름 | 종류 | 설명 |
|------|------|------|
| `grid` | 상수 (`number`) | 기본 그리드 단위 |

## 상세 명세

### `grid`

- 타입: `number`
- 값: `4`
- 역할: 레이아웃 간격·크기 계산의 기본 단위. 4px 기반 그리드 시스템의 기준값이다.

## 동작 흐름

모듈 임포트 시 `grid = 4` 상수가 즉시 평가되어 내보내진다. 별도의 런타임 로직이 없으며 순수하게 값을 공급하는 역할만 한다.
