# renderers/react/src/index.ts

## 개요

패키지의 메인 진입점으로, `v0_8` 서브모듈의 모든 공개 심벌을 투명하게 재수출한다. 사용자는 이 파일 하나를 통해 `@a2ui/react` 패키지의 모든 컴포넌트, 훅, 유틸리티, 타입에 접근할 수 있다. 현재 버전(`v0_8`)이 기본 버전으로 설정된다.

## 의존성

### 외부 패키지
없음.

### 저장소 내부 모듈
- [`./v0_8/index`](./v0_8/index.ts.md) — v0_8 서브모듈 전체 공개 API

## Exports

`export * from './v0_8/index'` 구문을 통해 `./v0_8/index`가 내보내는 모든 심벌을 그대로 재수출한다. 구체적인 내용은 `v0_8/index.ts`를 참조한다.

## 동작 흐름

파일은 단일 `export *` 구문만 포함한다. 직접적인 로직 없이 `v0_8/index`로 완전히 위임한다.
